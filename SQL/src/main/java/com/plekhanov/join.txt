(inner) join - соединяет строки двух таблиц по предикату <алиас1.колонка> = <алиас2.колонка>

left/right (outer) join - соединяет строки двух таблиц по предикату <алиас1.колонка> = <алиас2.колонка>, добавляет
    строки левой/правой таблицы, не совпавшие по предикату

full (outer) join - соединяет строки двух таблиц по предикату <алиас1.колонка> = <алиас2.колонка>, добавляет строки
    левой и правой таблицы, не совпавшие по предикату
